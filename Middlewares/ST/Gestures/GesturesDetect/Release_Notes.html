Release Notes for GesturesDetect Library

Copyright 2019 STMicroelectronics

 

Update History

V1.4.1 / 6-May-19

Main Changes

  • Minor issue fixed

V1.4.0 / 06-June-2016

Main Changes

  • Upgraded TAP_1 module to remove possible false detections when hand is entering in the sensor FoV and a background target is detected. New parameter added in the Init function : MinBackgroundDistance set to 300 mm

V1.3.0 / 13-May-2016

Main Changes

  • Added new parameter (handMustCoverBoth) to DIRSWIPE_1 module: set to false if ranging devices are far away from each other.
  • Added new ToF gestures modules : TAPS_1 to detect double taps (on a single device) and DIRSWIPE_TAP_1 to combine DIRSWIPE_1 and TAPS_1 modules

 

V1.2.0 / 15-April-2016

Main Changes

  • Improved TAP_1 to be able to work even if a background object is detected
  • Added new module (SWIPE_1) doing only swipe detection

 

V1.1.0 / 25-Nov-2015

Main Changes

  • Added new tof_gestures module called tof_gestures_TAP_1 and focusing only on TAP detection (more robust than TAP_SWIPE_2). This module is able to detect a TAP even if the gesture start is detected by the ToF device

 

V1.0.0 / 28-May-2015

Main Changes

  • Initial release.
